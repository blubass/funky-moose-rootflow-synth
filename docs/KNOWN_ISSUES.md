# Known Issues

We strive to make RootFlow as stable as possible, but like any evolving organism, there might be some quirks.

## General
- **macOS Builds are Unsigned**: You may need to manually allow the plugin in System Settings.
- **High CPU on Legacy Hardware**: 32-voice polyphony with high modulation density can be demanding on older processors.

## DAW Compatibility
- **Rescan Required**: Some DAWs (like Ableton Live or Logic Pro) might require a full plugin rescan if you replace an existing RootFlow build with a newer version.
- **AU Validation**: Audio Unit validation behavior may differ slightly between different macOS versions. If validation fails, try restarting your DAW.

## UI
- **Window Scaling**: On very small displays, the initial window size might feel large. You can resize the window manually.
